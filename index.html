Here are some of my projects

  • WeConnect

    MongoDB, Express, Node.js, React.js, Socket.io, Material UI

    WeConnect is a dynamic messaging web application built using the MERN stack. It offers users the ability to create personalized profiles and connect with friends through messaging. Upon registering, users can set up a profile with details such as name and profile picture. They can then search for and add friends within the platform. WeConnect facilitates real-time messaging between friends, enabling users to engage in private conversations. The app aims to provide a seamless and user-friendly experience for individuals looking to connect and communicate with their network.

  • Mini Messsaging Board

    MongoDB, Express, Node.js, PUG

    Developed a mini messaging board using Node.js, Express.js, JavaScript, and the Pug viewing template. This project follows the Model-View-Controller (MVC) architectural pattern, ensuring a structured and maintainable codebase. The messaging board allows users to post messages, view existing messages, and engage in real-time communication. Leveraging Node.js for backend logic and Express.js for routing, the application delivers seamless user experience while maintaining scalability.

  • E-Commerce Site

    JavaScript, React, Tailwind CSS, Fake Store API, React Router DOM

    This website uses JavaScript, React, and Tailwind CSS to design and implement a dynamic e-commerce website. Utilized the React Router DOM for seamless navigation, ensuring a smooth and intuitive user experience. Integrates the Fake Store API allowed to populate the website with realistic product data.

  • Memory Card Game

    HTML, CSS, JavaScript, React, Local Storage

    This web app uses a mock character API as cards for a memory game. Implemented hooks to store character data and to set the scores. Uses simple animation to show a shuffling effect for the cards. Stores the best score into local storage.

  • Weather App

    HTML, CSS, JavaScript, Weather API, npm

    Designed and developed a user-friendly weather application that provides real-time weather data using the WeatherAPI. Use the form to search for the current weather and the forecast of the next two days.

  • Battleship

    HTML, CSS, JavaScript, npm, Jest

    Place your ships on your board. When you're done, the game starts. Take turns with the computer to attack the opponent's board. First one to eliminate all 5 of the opponent's ships wins. Uses Jest to test the code.
