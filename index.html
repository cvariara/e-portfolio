Here are some of my projects

  • Twitter Clone App

    HTML, CSS, JavaScript, React (+hooks), Firebase

    This web app is a clone based of the social media app Twitter. Uses Material UI for the buttons and icons. Tweets are stored in Firebase in real time with the Firestore Database. On the right side, only 'Home' is functioning currently. Styled using HTML5 and CSS BEM best practices.

  • Weather App

    HTML, CSS, JavaScript, Weather API, npm

    Designed and developed a user-friendly weather application that provides real-time weather data using the WeatherAPI. Use the form to search for the current weather and the forecast of the next two days.

  • Battleship

    HTML, CSS, JavaScript, npm, Jest

    Place your ships on your board. When you're done, the game starts. Take turns with the computer to attack the opponent's board. First one to eliminate all 5 of the opponent's ships wins. Uses Jest to test the code.

  • To-Do List

    HTML, CSS, JavaScript, Local Storage

    Tasks are stored in Projects. Start off by making a Project which can then add Tasks. Setup Tasks with a title, due date, and whether it is important. The tabs on the sidebar filters the tasks with their dates. Tasks and Projects are stored in Local Storage.

  • Library App

    HTML, CSS, JavaScript, React (+hooks)

    This web app pulls data from a mock book API which you can browse. Uses Font Awesome for icons. In the all books section, you can sort the books by rating or price. There is a cart implemented that stores the data of the books added. Uses React Router to navigate between the different tabs.

  • Pokémon Clone Game

    HTML, CSS, JavaScript

    This web game is a clone of the popular Pokémon series. You move around the island ad the character and can encounter monsters in the dark patches of grass. The game has audio for different scenarios and uses the howler library. The battle animations are being made with the GSAP library. All the sprites and tiles are taken from itch.io which is free to use.

  • Movie App

    HTML, CSS, JavaScript

    This web app pulls from The Movie Database API and displays all the latest movies on the screen. There is a search engine implemented that can look up any movie in the api. You can add movies to a watch list by clicking on the button when you hover over the movie.
